需求

  • 树莓派 4，microSD 卡， 电源线，网线(或显示器，键盘，MicroHDMI 线)
  • Ubuntu 镜像: https://ubuntu.com/download/raspberry-pi
  • Win32 Disk Imager: https://sourceforge.net/projects/win32diskimager/
  • 格式化 microSD 卡，如果 microSD 卡有多个分区，用磁盘管理工具合并

向 microSD 卡写入系统

Windows 电脑打开 Win32 Disk Imager，选择映像文件与设备，点击写入，结束后弹出的询问是否格式化窗口点取消

插入树莓派后接通电源，等待 5 分钟
