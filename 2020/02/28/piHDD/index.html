准备

硬盘使用使用高品质有源硬盘盒，并格式化为 ext4 减少 bug ，提升性能

插入硬盘后

确定挂载路径如 /mnt 并赋予权限

挂载硬盘

设置开机自动挂载

因为连接问题或重启后 /dev 的路径偶尔会发生变化，所以按照 PARTUUID 设置挂载路径

添加至 fstab

部分参数解释：

  • nofail：硬盘连接时挂载，未连接时跳过，防止未连接时无法开机

  • x-systemd.device-timeout：与 nofail 搭配，未连接时默认等待时间为 90 秒，设置后减少等待时间，但不能为 0 或过短，否则会无限等待或跳过自动挂载造成自动挂载失败，经测试设置为 5 时可能跳过挂载

  • noatime：阻止多余的写操作，提升性能
